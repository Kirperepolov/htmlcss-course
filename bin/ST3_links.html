All Autocomplete

Первый в списке musthave плагин. Анализирует все открытые в Sublime файлы и файлы из папок Вашего проекта, и при написании кода предлагает полный список автодополнений (аналогично автодополнению в Visual Studio) Установка: ctrl+shift+p > Install Packages > All Autocomplete

BracketHighlighter

Плагин, заметно и наглядно демонстрирующий, в какой секции кода Вы находитесь. Распознает все виды скобок и кавычек, а так же секции #ifdef и еще несколько специфических синтаксических конструкций. Установка: ctrl+shift+p > Install Packages > BracketHighlighter

DoxyDoc

Набор сниппетов по созданию Doxygen-совместимой документации. Работает для языков C/C++/C#/Obj-C, Java, JavaScript. Сниппеты построены на JavaDoc-командах, в них легко разобраться и без оф. документации. Не разобрались? Курите матчасть. Вставка документации для написанной функции происходит после написания непосредственно перед ней кода /** и нажатия Tab Установка: ctrl+shift+p > Install Packages > DoxyDoc

Git
Ну, сложно не догадаться, что делает этот плагин. Управляйте своим репозиторием прямо из Sublime (да, прямо как в БОЛЬШИХ УБЕРДОРОГИХ IDE ОТ КОКОСОФТ!!1!1) Для взаимодействия с самим гитом есть пара-тройка хоткеев, да и вообще все очень просто и хорошо сделано. Установка: ctrl+shift+p > Install Packages > Git
GitGutter

Надстройка над стандартным Git, добавляющая всяких визуальных свистелок-перделок а-ля индикации добавленных/изменённых/удалённых строк в текущем/предыдущих коммитах, демонстрации diff’а за текущий коммит и быструю навигацию между внесенными изменениями. Есть всякие там цветовые схемы и возможность их кастомизации. Качайте, короче. Установка: ctrl+shift+p > Install Packages > GitGutter

SublimeLinter

Ну, по этому плагину (ПЛАГИНИЩЩУ) я проехался в своей предыдущей статье вдоль и поперёк. И наискось, наверное, тоже. Это — плагин, без которого вы вряд ли сможете жить и юзать Sublime для разработки кода. БЫСCСТРА устанавливать его и настраивать так, как написано вот тут.

Установка: ctrl+shift+p > Install Packages > SublimeLinter, ctrl+shift+p > Install Packages > SublimeLinter-contrib-clang

Side​Bar​Enhancements

Плагин, заметно расширяющий функционал контекстного меню файлов в боковой панели. Тут вам и стандартные операции с файлами, и стандартные операции с папками, и работа с проектами Sublime, и «Открыть в Проводнике»… Очень нужный плагин, в общем. Установка: ctrl+shift+p > Install Packages > Side​Bar​Enhancements
